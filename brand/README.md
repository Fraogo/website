# Fraogo Brand Assets

Single source of truth for Fraogo's visual identity. All frontend work must align with these.

| File | Inside |
|------|--------|
| [colors.md](colors.md) | Palette — primary, neutrals, semantic, gradients |
| [typography.md](typography.md) | Font, weights, size scale, rules |
| [logo.md](logo.md) | Variants, placement, clearspace, don'ts |
| [voice.md](voice.md) | Voice, tone, writing style |

**Logo files** live in `public/logo/`: `logo.png` (light bg) and `logo-white.png` (dark bg).

> Current files are `.jpeg` — rename/replace with `.png`. Code expects `.png`.
