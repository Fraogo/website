# Fraogo Brand Assets

This folder is the single source of truth for Fraogo's visual identity.
All frontend work must align with what's defined here.

## Contents

| File | What's inside |
|------|--------------|
| [colors.md](colors.md) | Full colour palette — primary, neutrals, semantic colours |
| [typography.md](typography.md) | Font family, weights, size scale, usage rules |
| [logo.md](logo.md) | Logo variants, placement rules, clearspace, don'ts |
| [voice.md](voice.md) | Brand voice, tone, writing style |

## Logo Files

Logo image files live in `public/logo/`:

| File | Use when |
|------|----------|
| `logo.png` | Light backgrounds, white pages, cards |
| `logo-white.png` | Dark backgrounds, hero sections, dark navbars |

> **Note:** Current files in `public/logo/` are `.jpeg` — rename or replace with `.png` versions. The site code expects `.png`.
